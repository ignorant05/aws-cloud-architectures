# aws-cloud-architectures

## Description
This repository is my own aws cloud architectures/designs for several applications

## About 
- Each sub-directory corresponds to a standalone architecture
- Each architecture's IaC with **Terraform** contains it's own `README.md` files (one for each module), each generated by [terraform-docs](https://github.com/terraform-docs/terraform-docs.gi).
- Each root directory contains an overview of the overall architecture + diagram(s)
